# The Shell Journey Offensive Security Edition

Welcome to my shell scripting repository. This project serves as a digital log of my journey from shell foundations to advanced malware development and exploit research.

## Project Mission
I believe that high quality technical education should be accessible to everyone. In this repository, I recreate and document core shell scripting concepts taught in paid courses, providing original examples and scripts through the lens of a security researcher. This is for those who may not have the resources to purchase formal training.

## Repository Map

### 1 Foundations
Core concepts including variables, loops, and conditional logic, applied to security automation.

### 2 File Systems and RE
Scripts for navigating Linux file systems, finding sensitive data, and assisting in Windows or Linux reverse engineering workflows.

### 3 Red Team Automation
Automated reconnaissance, system enumeration, and lateral movement scripts.

### 4 Malware Dev and Persistence
Research into Linux malware techniques, shell based persistence, and payload delivery.

## How to Use
1 Clone the repository to your local machine.
2 Navigate to the desired folder.
3 Grant execution permissions using chmod plus x scriptname.sh.
4 Run the script using dot slash scriptname.sh.
